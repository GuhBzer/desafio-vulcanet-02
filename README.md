## :computer: Projeto

  Projeto simula o fluxo de algumas das funcionalidades (Whatsapp e Email) do Dashboard do peçaZap.

## :information_source: Como executar

### Requerimentos

Para rodar a aplicação você vai precisar instalar:
* [Git](https://git-scm.com)
* [Node](https://nodejs.org/)
* [Yarn](https://yarnpkg.com/)


### Iniciando a aplicação
Clone o repositório e instale as dependências.
```bash
# clone o repositório
$ git clone https://github.com/StefanoSaffran/desafio-vulcanet-02.git

# acesse a pasta do projeto
$ cd desafio-vulcanet-02/

# instale as dependências
$ yarn

# inicie a aplicação
$ yarn start

```
